### Cambiar Información del Negocio
Edita `src/config/siteConfig.js`:

```javascript
business: {
  name: "Tu Academia",
  tagline: "Tu Tagline",
  phone: "+54 11 1234-5678",
  email: "info@tuacademia.com",
  address: "Tu dirección"
```
